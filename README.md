# Raspberry Pi Web Hosting using ngrok

This project demonstrates how to host a local web server on Raspberry Pi
and expose it to the internet using ngrok.

## Technologies Used
- Raspberry Pi
- Python HTTP Server
- ngrok

## Steps
1. Start the Python web server
2. Run ngrok to expose port 80
3. Use the public URL generated by ngrok

## Run Server
sudo python3 server.py

## Start ngrok
ngrok http 80
